# 5.字符串与指针
**字符数组没有'/0'，全是字符''**


```c++
char c[] = {'c'}
char c[] = 'china'
```
前者是字符数组，后者是字符串



```c++
char a[10];
char *p;
p = a;
```
**此时**

```c++
cout << p << endl;
```
直接输出到最后


```c++
cout << p++ << endl;
```
输出第二个字符到最后

**一维数组输出的时候是地址**
**字符数组输出的时候是字符，除非用static_cast<*void>()**


```c++
cout << *p << endl;
```
输出指向的第二个字符

## 指向二维数组指针


```c++
int a[3][4] = {} 
int b[3] = {}
# 假定都赋值有实际内容
```
&上升一个层面
*下降一个层面

**只是指向首元素和上一级整体的时候通常一致**


二维数组的指针是a[0], a[1], a[2]



**&a
a
a[0]
a[0][0]
可以通过*与&互相等价**

*(a+1)在二维数组里面实则还是指针
