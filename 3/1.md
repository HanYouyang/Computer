
## 2.2引用

```c++
int & r = n;
```

不能引用常量，赋值则引用与被引用全部改变

也不能引用表达式

好处在于，函数调用的时候取地址&引用，可以代替用指针作为参数

const 常引用则不能被修改

## 3.1关键字和常量

const 定义各种类型的常量

常量指针也不能被修改，指向的东西可以被修改

常量指针不能被赋值给变量指针

## 3.2动态内存分配

new 动态分配

```c++ 
P = new T;
P = new T[N];
```

返回值类型都是T*

则左边就应该是相应的类型

```c++
delete p;

```

## 3.3内联函数函数重载

调用的额外开销，设计内联函数

将函数的代码插入到函数体中

```c++
inline int max(int a, int b){}
```

函数重载，只要参数表不一样，名字一样就无所谓

参数表当然包括，数量、类型
